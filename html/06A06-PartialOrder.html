partial order


A partial order (often simply referred to as an order or ordering) is a relation ≤⊂A×A that satisfies the following three properties:

  1. 1.

    Reflexivity: a≤a for all a∈A

  2. 2.

    Antisymmetry: If a≤b and b≤a for any a,b∈A, then a=b

  3. 3.

    Transitivity: If a≤b and b≤c for any a,b,c∈A, then a≤c

A total order is a partial order that satisfies a fourth property known as comparability:

  • •

    Comparability: For any a,b∈A, either a≤b or b≤a.

A set and a partial order on that set define a poset.

Remark. In some literature, especially those dealing with the foundations of mathematics, a partial order ≤ is defined as a transitive irreflexive binary relation (on a set). As a result, if a≤b, then b≰a, and therefore ≤ is antisymmetric.
