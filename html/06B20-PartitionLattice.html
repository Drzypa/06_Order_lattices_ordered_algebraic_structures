partition lattice


The partition lattice (or lattice of partitions) Πn is the lattice of set partitions (http://planetmath.org/Partition) of the set [n]={1,…,n}. The partial order on Πn is defined by refinement, setting x≤y if any only if each cell of x is contained in a cell of y.
