locally finite poset


A poset P is locally finite if every interval [x,y] in P is finite. For example, ℤ with the usual order is locally finite but not finite, while ℚ is neither.

Every locally finite poset is also chain finite, but the converse does not hold. To see this, define a partial order on ℕ by the rule that k≤ℓ if and only if k=0 or ℓ=1. Thus 0 is the minimum element, 1 is the maximum element, and the remaining elements form an infinite antichain. Every bounded chain in this poset is finite but the entire poset is an infinite interval, so the poset is chain finite but not locally finite.
